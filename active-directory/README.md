# Active Directory Home Lab (Windows Server 2022)

## Overview

This section documents my hands-on Active Directory home lab built to simulate real-world Help Desk support tasks. The lab focuses on user account management, domain authentication, basic troubleshooting, and patch management.

## Lab Environment

- Hypervisor: Oracle VirtualBox
- Network Type: NAT Network
- NAT Network Name: Null
- Domain Name: Ayowale.com

### Virtual Machines
- Windows Server 2022 (Domain Controller)
- Windows 11 Pro N (Domain-joined client)

## Documentation Sections

- Setup:
  - Windows Server 2022 AD DS installation
  - Windows 11 domain join
- User Management:
  - Creating and managing Active Directory users
- Troubleshooting:
  - Locked account simulation and resolution
- Patch Management:
  - Patch management and vulnerability remediation using Action1
