# Virtual IT Support Home Lab

This repository documents a comprehensive virtualized IT support environment built using Oracle VirtualBox to simulate real-world enterprise infrastructure.

## Lab Environment
* *Hypervisor:* VirtualBox 7.1.10
* *Infrastructure:* * Windows Server 2022 (Domain Controller)
    * Windows 10 Enterprise (Workstation)
    * Ubuntu & Kali Linux (Network Testing)

## Project Objectives & Skills
* *Active Directory Management:* Deploying AD DS, managing User Accounts, and resolving account lockouts or password resets.
* *OS Deployment:* Installation and configuration of enterprise Windows and Linux environments.
* *Networking:* Configuring NAT networking and verifying DNS connectivity between clients and the DC.
* *Technical Documentation:* Maintaining detailed workflows for system setups and incident resolution.

## Repository Structure
* /active-directory: User management and Group Policy documentation.
* /architecture: High-level overview of the lab network.
* /assets: Screenshots and visual evidence of lab configurations.
